# BitPacker: Visual Game Netcode Designer & Bandwidth Simulator

[English README](./README.md) | [Русское руководство](./README.ru.md)

**BitPacker** — это визуальный веб-инструмент на базе React и TypeScript для проектирования битовых структур сетевых пакетов и точной симуляции трафика. Он разработан специально для многопользовательских проектов на платформах **Roblox (Luau)**, **s&box (C#)** и **Unity (Netcode for GameObjects)**.

Инструмент позволяет превратить разрозненные сетевые переменные в плотно упакованный двоичный буфер, сводя к минимуму накладные расходы протоколов и экономя до **90% исходящего трафика сервера**.

---

## ⚡ Ключевые возможности

1. **Конструктор сетевых пакетов (Schema Designer)**
   * Добавление переменных (`int`, `bool`, `float`, `Vector3`, `string`).
   * Настройка битового сжатия:
     * Ограничение диапазона целых чисел (`minValue`/`maxValue`) для автоматического расчета необходимого количества бит.
     * Квантование чисел с плавающей точкой (`float`) и векторов (`Vector3`) по заданным границам и шагу точности.
     * Оптимальные динамические и фиксированные строки UTF-8.
     * Защита от использования зарезервированных ключевых слов.

2. **Интерактивный визуализатор сетки бит (Bit-Layout Visualizer)**
   * Графическое отображение распределения переменных в байтах.
   * Визуальное разделение полезной нагрузки, битов выравнивания (`padding`) и свободного места.
   * Инспекция каждого бита и его смещения при наведении.
   * Предупреждения о неэффективном использовании памяти.

3. **Симулятор сетевого трафика (Bandwidth Simulator)**
   * Анализ загрузки сети с учетом накладных расходов популярных сетевых оболочек и протоколов (Roblox RemoteEvent, Unity UTP, s&box Steamworks, Raw UDP, TCP, WebSocket).
   * Полная математическая модель репликации: расчет квадратичного масштабирования трафика рассылки сервера $O(N \cdot (N - 1))$.
   * Динамический SVG-график сравнения трафика: BitPacked vs Raw Replica vs JSON String.

4. **Генератор готового кода сериализации (Code Generator)**
   * Компиляция в реальном времени.
   * Выходной код является синтаксически валидным и оптимизированным под целевую платформу:
     * **Roblox (Luau):** Использование Luau buffer API (`buffer.writeu8`, `buffer.writestring` и т.д.) с встроенными функциями сжатия half-precision float (16 бит) без потери данных.
     * **s&box (C#):** Потоковая сериализация через `BinaryWriter` и `BinaryReader` с защитой от переполнения UTF-8 буфера.
     * **Unity (NGO C#):** Реализация интерфейса `INetworkSerializable` с использованием `FastBufferWriter`/`FastBufferReader` и автоматическим маппингом фиксированных строк (`FixedString32Bytes`, `FixedString64Bytes` и т.д.).

## 🚀 Быстрый запуск

### Требования
* Установленный **Node.js** (версии 18 или новее)

### Установка зависимостей
Склонируйте репозиторий и установите пакеты:
```bash
npm install
```

### Запуск сервера разработки
Запустите локальный сервер Vite:
```bash
npm run dev
```
Откройте браузер по адресу `http://localhost:5173/`.

### Запуск Unit-тестов
Запустите тесты для проверки алгоритмов упаковки, расчетов симулятора и генераторов кода:
```bash
npm test
```
